# Release Notes

## v2.0.0 - DEEVO Cortex Launch (March 2026)

### 🌟 Highlights

- **DEEVO GCC Command Center** - World-class decision intelligence surface
- **Real-time Signal Processing** - Live signal ingestion and classification
- **Causal Graph Engine** - Model complex relationships and propagate impacts
- **Decision Intelligence** - AI-powered APPROVE/REVIEW/ESCALATE recommendations
- **GCC Focus** - Purpose-built for Gulf Cooperation Council markets

### ✨ Features

#### Command Center
- Interactive GCC globe with country-specific positioning
- Real-time signal feed with severity indicators
- Decision panel with confidence scores and reasoning
- Executive brief panel for quick summaries
- Propagation trace visualization
- Country highlight cards

#### Signal Processing
- Multi-source signal ingestion (MACRO, MARKET, INSURANCE, FRAUD, REGULATORY)
- Automatic severity classification
- Trend analysis and anomaly detection
- GCC-specific signal sources

#### Graph Engine
- Causal relationship modeling
- Impact propagation with weighted edges
- Interactive node exploration
- Real-time graph updates

#### Decision Engine
- Rule-based evaluation
- Confidence scoring
- Risk assessment
- Audit trail generation

#### DEEVO Monitor (v2.1)
- Live decision intelligence operating system at `/monitor`
- Causal Wave Engine with requestAnimationFrame particle simulation
- Cognitive Scenario Engine — multi-path futures with probability scoring
- Pre-Causal Engine — pressure field model detecting risk before signals form
- Scenario Video Engine — cinematic 15s video generation (1080×1920 / 1920×1080)
- 5 interactive view modes: GCC Map, Wave Sim, Cognitive, Pre-Causal, Video

### 🛠️ Technical

- React 18 + Vite + TypeScript + Tailwind CSS
- useReducer + Context for central state (no Redux)
- FastAPI backend (optional — demo fallback built in)
- HTML5 Canvas + SVG visualizations
- MediaRecorder video export (H.264/VP9)
